# <a name="azure-data-factory-connector-overview"></a>Visão geral do conector do Azure Data Factory

Azure Data Factory oferece suporte aos seguintes armazenamentos de dados por meio de atividade de cópia, fluxo de dados, atividade de pesquisa, atividade de obtenção de metadados e atividade de exclusão. Clique em cada armazenamento de dados para aprender os recursos com suporte em detalhes e as configurações correspondentes.
