# <a name="troubleshoot-a-declined-card-at-azure-sign-up"></a>Solucionar problemas de um cartão recusado na inscrição do Azure

Este artigo ajuda você a resolver problemas em que um cartão de crédito é recusado na inscrição do Azure no portal do Azure ou no centro de contas do Azure. Antes de começar a solucionar o problema, verifique os seguintes pontos:

- Verifique se as informações fornecidas para seu perfil de conta do Azure, como email de contato, endereço e número de telefone, estão corretas.
- Verifique se as informações do cartão de crédito estão corretas.
- Verifique se você já não tem uma conta da Microsoft com as mesmas informações.
- Os cartões de débito não são aceitos

## <a name="issues"></a>Problemas

A lista a seguir contém problemas comuns que podem fazer com que um cartão de crédito seja recusado na inscrição do Azure.

### <a name="the-credit-card-provider-is-not-accepted-for-your-country"></a>O provedor de cartão de crédito não é aceito para seu país

Quando você escolhe um cartão, o Azure exibe as opções de cartão que são válidas no país selecionado. Contate o seu banco ou o emissor do cartão para verificar se o seu cartão de crédito está habilitado para transações internacionais. Para obter mais informações sobre países e moedas com suporte, consulte as [perguntas frequentes sobre compra do Azure](https://azure.microsoft.com/pricing/faq/).

>[!Note]
>Os cartões de crédito American Express não têm suporte no momento como um instrumento de pagamento na Índia. Não temos um período de tempo para quando pode ser uma forma de pagamento aceita.

### <a name="youre-using-a-virtual-or-prepaid-card"></a>Você está usando um cartão pré-pago ou virtual 

Cartões de crédito ou débito pré-pago ou virtuais não são aceitos como pagamentos de assinaturas do Azure.

### <a name="your-credit-information-is-inaccurate-or-incomplete"></a>Suas informações de crédito estão incorretas ou incompletas 

O nome, o endereço e o código CVV inseridos devem corresponder exatamente ao que é impresso no cartão.

### <a name="the-card-is-inactive-or-blocked"></a>O cartão está inativo ou bloqueado 

Entre em contato com seu banco para verificar se o cartão está ativo.

Você pode estar enfrentando outros problemas de inscrição 

Para obter mais informações sobre como solucionar problemas de inscrição do Azure, consulte o seguinte artigo da base de dados de conhecimento: 

[Você não pode se inscrever no Azure no portal do Azure ou no centro de contas do Azure](billing-troubleshoot-azure-sign-up.md)

### <a name="you-represent-a-business-that-doesnt-want-to-pay-by-card"></a>Você representa um negócio que não deseja pagar por cartão 

Se você representar um negócio, poderá usar métodos de pagamento de faturas, como cheques, cheques noturnos ou transferências de transmissão para pagar pela sua assinatura do Azure. Depois de configurar a conta para pagar por fatura, você não poderá mudar para outra opção de pagamento, a menos que tenha um contrato de cliente da Microsoft e se inscrever no Azure por meio do site do Azure.

Para obter mais informações sobre como pagar por fatura, consulte [Enviar uma solicitação para pagar a assinatura do Azure por fatura](billing-how-to-pay-by-invoice.md).

### <a name="your-credit-card-information-is-outdated"></a>Suas informações de cartão de crédito estão desatualizadas 

Para obter informações sobre como gerenciar as informações do cartão, incluindo a alteração ou a remoção de um cartão, consulte [Adicionar, atualizar ou remover um crédito do Azure](billing-how-to-change-credit-card.md).
