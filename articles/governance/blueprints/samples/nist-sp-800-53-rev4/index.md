# <a name="overview-of-the-nist-sp-800-53-r4-blueprint-sample"></a>Visão geral da amostra de blueprint NIST SP 800-53 R4

A amostra de blueprint NIST SP 800-53 R4 oferece proteções de governança usando o [Azure Policy](../../../policy/overview.md), as quais ajudam a avaliar controles NIST SP 800-53 R4 específicos. Esse blueprint ajuda os clientes a implantar um conjunto principal de políticas para qualquer arquitetura implantada pelo Azure que deve implementar os controles NIST SP 800-53 R4.

## <a name="control-mapping"></a>Mapeamento de controle

A seção de mapeamento de controle oferece detalhes sobre políticas incluídas dentro desse blueprint e como elas abordam vários controles no NIST SP 800-53 R4. Quando atribuídos a uma arquitetura, os recursos são avaliados pelo Azure Policy para verificar a não conformidade com políticas atribuídas. Para saber mais, veja [Azure Policy](../../../policy/overview.md).
