# <a name="azure-cli-samples-for-azure-cosmos-db-sql-core-api"></a>Exemplos da CLI do Azure para a API do SQL (Core) do Azure Cosmos DB

A tabela a seguir inclui links para exemplos de scripts de CLI do Azure para a API do SQL (Core) do Azure Cosmos DB. As páginas de referência de todos os comandos do Azure Cosmos DB CLI estão disponíveis na [Referência de CLI do Azure](/cli/azure/cosmosdb). Para obter exemplos de CLI do Azure para outras APIs do Azure Cosmos DB, confira [API do Cassandra](cli-samples-cassandra.md), [API do MongoDB](cli-samples-mongodb.md), [API do Gremlin](cli-samples-gremlin.md) e [API de Tabela](cli-samples-table.md). Todos os exemplos de scripts da CLI do Azure Cosmos DB podem ser encontrados no [Repositório GitHub da CLI do Azure Cosmos DB](https://github.com/Azure-Samples/azure-cli-samples/tree/master/cosmosdb).
