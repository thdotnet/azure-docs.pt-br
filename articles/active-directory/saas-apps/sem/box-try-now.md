# <a name="box-and-azure-active-directory"></a>Box e Azure Active Directory

Acelere a adoção do box em sua organização e aumente a adoção do usuário com nossa integração predefinida. Permita que os usuários colaborem rapidamente conectando a caixa ao Azure Active Directory (Azure AD).

![Mostra o logotipo do aplicativo Box](./media/box.png)

> [!div class="nextstepaction"]
> [Caixa de tentativa com o Azure AD >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>O que você pode fazer com o box e o Azure AD?

- **Logon único fácil** -elimine vários nomes de usuário e senhas e simplifique o acesso permitindo que os usuários entrem no box usando suas contas do Azure AD.
- **Automatizar o provisionamento de usuários** – Automatize a criação, a manutenção e a remoção de contas de usuário no box quando os usuários ingressam, movem ou deixam sua organização.
- **Proteja, administre e monitore** o acesso seguro ao box aplicando políticas de acesso condicional, administre o acesso por meio de revisões de acesso e monitore o acesso com logs de entrada.

## <a name="get-started"></a>Introdução

Configure o [provisionamento de usuário](https://aka.ms/BoxSSODoc) e [logon único](https://aka.ms/BoxProvisioningDoc) para o box com nossos guias de configuração fáceis.
